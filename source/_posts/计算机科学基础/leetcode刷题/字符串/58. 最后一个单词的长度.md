> Problem: [58. 最后一个单词的长度](https://leetcode.cn/problems/length-of-last-word/description/)

  

# 思路

  

评价为随便过

  

字符串去掉前后空格，再从后往前找到最后一个单词，注意：不要越界，判断i >= 0

  

# 复杂度

  

时间复杂度:

> 添加时间复杂度, 示例： $O(n)$

  

空间复杂度:

> 添加空间复杂度, 示例： $O(n)$

  
  
  

# Code

```C++ []

class Solution {

public:

    int lengthOfLastWord(string s) {

        trim(s);

        int i = s.size() - 1;

        int j = s.size() - 1;

        while (i >= 0 && s[i] != ' ')

            i--;

        return j - i + 1;        

    }

  

    void trim(string& s) {

        int j = s.size() - 1;

        while (s[j] == ' ')

            j--;

        int i = 0;

        while (s[i] == ' ')

            i++;

        s.substr(i, j - 1 + 1);

        s.resize(j - 1 + 1);

    }

};

```
