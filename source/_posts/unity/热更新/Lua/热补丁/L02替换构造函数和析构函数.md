### 先写一个C#的带有构造和析构函数的类用以替换

```cs
[Hotfix]
public class TestHotFix
{
    public TestHotFix()
    {
        Debug.Log("C#构造函数");
    }
    ~TestHotFix()
    {
        Debug.Log("C#析构函数");
    }
}
```

### 固定的`lua`解析函数`.ctor`

在lua中用`.ctor`来代表解析函数

```lua
xlua.hotfix(CS.TestHotFix, {

    -- 固定lua解析函数 .ctor

    [".ctor"] = function()

       print("Lua构造函数")

    end
}
```

## 固定lua析构函数 Finalize

在lua中用`Finalize`来代表析构函数

```lua
xlua.hotfix(CS.TestHotFix, {

    Finalize = function()

        print("lua析构函数")

    end

})
```

## 注意

>   热补丁并不会替换解析函数和析构函数
>   
>   而是在C#调用之后再调用
